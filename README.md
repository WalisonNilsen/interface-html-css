# Projeto: Curriculo online

Projeto desenvolvido com foco em aprendizado de frontend, aplicando conceitos de estruturação de layout e interações visuais sem uso de JavaScript.

## Objetivo
Construir uma interface web moderna utilizando HTML e CSS, com foco em:
- Organização de layout com Flexbox
- Estrutura semântica de página
- Interações com hover e transições
- Separação entre estrutura e estilo

## Funcionalidades
- Navbar com indicação de página ativa
- Botões com feedback visual (hover + transição)
- Cards interativos com efeito de elevação
- Layout centralizado e organizado

## Tecnologias
- HTML
- CSS

## Aprendizados
Durante o desenvolvimento, pratiquei:
- Uso de `flexbox` para layout
- Diferença entre `margin` e `padding`
- Uso de `box-sizing: border-box`
- Interações com `transform` e `transition`
- Construção de componentes reutilizáveis
